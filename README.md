# 🌍 موقع أسعار الذهب العالمي - Gold Prices News

نظام مؤتمت ذكي لنشر أسعار الذهب العالمية بجميع اللغات والدول، مع كتابة مقالات تلقائية عند كل تغيير في السعر.

## ✨ المميزات

- 🤖 **نشر تلقائي للمقالات** عند أي تغيير في سعر الذهب
- 🌐 **دعم جميع اللغات** - 100+ لغة
- 🌍 **تغطية عالمية** - جميع الدول والمناطق
- 📰 **مقالات ذكية** مكتوبة بـ AI
- 🔍 **محسّن للـ SEO** لظهور أفضل في جوجل
- 💾 **قاعدة بيانات** لحفظ السجل التاريخي
- ⚡ **سرعة فائقة** - تحديثات لحظية
- 💰 **100% مجاني** - بدون تكاليف إضافية

## 🚀 كيفية البدء

### المتطلبات
- حساب GitHub (لديك ✅)
- حساب Vercel مجاني (لاستضافة الموقع)
- حساب Firebase أو MongoDB (قاعدة البيانات)

### التثبيت

1. استنسخ المستودع
```bash
git clone https://github.com/hassannasabx/gold-prices-news.git
cd gold-prices-news
```

2. ثبت المتطلبات
```bash
npm install
```

3. أنشئ ملف `.env` بـ:
```
FIREBASE_API_KEY=your_key
GOLD_API_KEY=your_key
```

4. شغّل المشروع محلياً
```bash
npm run dev
```

5. ارفعه على Vercel
- اذهب إلى https://vercel.com
- اختار المستودع
- انقر Deploy
